# Libux
A text-based interface application designed for basic configuration of a Linux server, specifically the Debian distribution.

## Features
- Firewall (iptables)
- Network connectivity
- DHCP
- SSH
- Apache2
- Routing
- Users and groups
- Disk quotas
- Permissions
- MariaDB

## Requirements
- Python 3.x
- Debian
- Root permissions

## Installation
```bash
git clone https://github.com/Forbelskym22/Libux.git
pip install -r requirements.txt
python main.py
```

## Usage
```bash
sudo python main.py
```
> Root permissions are required for most features.
